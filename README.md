## 🗃 テーブル設計（例）

### study_sessions テーブル

| カラム名      | 型        | 説明                   |
| ------------- | --------- | ---------------------- |
| id            | integer   | 主キー                 |
| user_id       | integer   | 紐づくユーザー ID      |
| date          | date      | 学習した日付           |
| start_time    | datetime  | 学習開始時間           |
| end_time      | datetime  | 学習終了時間           |
| total_minutes | integer   | 合計学習時間（分単位） |
| created_at    | timestamp | 作成日時               |
| updated_at    | timestamp | 更新日時               |
